# ML Model

The trained model is not included in this repo due to file size.

## To get the model:
1. Run `train_model.py` on your dataset
2. Or download from Google Colab output

## Model details:
- Base: bert-base-uncased
- Dataset: 16,466 samples
- Labels: confused, focused, annoyed, stressed, exhausted, 
          break, giveup, trying, needassistance, waiting
- F1 Score: ~0.78
